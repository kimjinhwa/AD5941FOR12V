### Test Method 
- BlueTooth를 연결한다. 
- Bat calibration을 하기위하여 runmode를 0으로 실행한다. 
- r 0 를 실행하여 Bat연결을 해제 한다. -> LED가 꺼진다.   
  LED는 0을 입력할시 꺼지면 BAT 선택한경우 켜진다.

- r 1 를 실행하여 Bat연결한다.
- cal를 수행한다.
- 정상적으로 수행이 되는 것을 확인하면, 다시한번 cal save를 입력한다.
